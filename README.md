OSIntegration
----------------

이 프로젝트는 오래된 여러 OS를 WIN32 환경에서 개발할 수 있도록 수정하는 것을 목표로 한다.  
커널 로딩은 GRUB과 SkyOS의 부트로더를 사용해서 부팅 프로세스를 단일화시킨다.
